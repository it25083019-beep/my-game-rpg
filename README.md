# README

このREADMEは、以下の2つのHTMLファイルについて説明します。

1. `janken_rpg_mario.html`
2. `weather-outfit-app.html`

どちらも、HTML / CSS / JavaScript を1つのHTMLファイル内にまとめたスマートフォン向けWebアプリです。

---

## 1. ジャンケンRPG WORLD

### ファイル名

`janken_rpg_mario.html`

### 概要

`ジャンケンRPG WORLD` は、ジャンケンのルールを使って敵と戦うレトロ風RPGゲームです。  
プレイヤーはキャラクターを選び、ステージごとに登場する敵とジャンケンバトルを行います。

勝つと敵にダメージを与え、負けると自分がダメージを受けます。  
ステージを進めると強い敵が登場し、最後にはボス戦があります。

### 主な機能

- タイトル画面
- キャラクター選択画面
- 複数ステージ制
- ボスバトル
- HPバー表示
- スコア表示
- ステージ進行バー
- アイテム機能
  - キノコ：HP回復
  - スター：必勝効果
  - フラワー：大回復
  - コウラ：守備力アップ
- ステータス画面
- 勝利 / 敗北画面
- レトロゲーム風のドットUI
- スマートフォン向けのタッチ操作

### 操作方法

1. `START GAME` を押します。
2. キャラクターを選びます。
3. `DECIDE!` を押してゲームを開始します。
4. バトル中は以下の3つから選びます。
   - グー
   - パー
   - チョキ
5. 敵に勝つと次のステージへ進みます。
6. 最後のボスを倒すとクリアです。

### ゲームルール

| 結果 | 内容 |
|---|---|
| 勝ち | 敵にダメージ |
| 負け | プレイヤーがダメージ |
| あいこ | ダメージなし |

ジャンケンの基本ルールは以下の通りです。

| 手 | 勝てる相手 |
|---|---|
| グー | チョキ |
| チョキ | パー |
| パー | グー |

### 使用技術

- HTML
- CSS
- JavaScript
- SVG
- CSS Animation
- レスポンシブデザイン

### 特徴

このゲームは外部のゲームエンジンやライブラリを使わず、1つのHTMLファイルだけで動作します。  
スマートフォンの画面でも遊びやすいように、ボタンを大きくし、縦画面で見やすいレイアウトにしています。

### 実行方法

1. `janken_rpg_mario.html` をダウンロードします。
2. ブラウザでファイルを開きます。
3. そのままゲームをプレイできます。

## 2. 今日のコーデ予報

### ファイル名

`weather-outfit-app.html`

### 概要

`今日のコーデ予報` は、天気や気温に合わせて服装を提案するスマートフォン向けWebアプリです。  
位置情報を取得し、天気データをもとに、その日のコーディネートを表示します。

APIキーが設定されていない場合でも、デモ用の天気データで動作します。

### 主な機能

- 現在地の取得
- 天気データの表示
- 気温、体感温度、湿度、風速、視程の表示
- 時間帯別の天気表示
- 天気に合わせた服装提案
- コーディネート詳細のボトムシート表示
- コーデのシャッフル
- コーデの保存ボタン
- 天気テーマ切り替え
  - 晴れ
  - 雨
  - 曇り
  - 寒い
  - 嵐
- 天気に応じた背景アニメーション
- スマートフォン向けUI

### 画面構成

| 画面 / 部分 | 内容 |
|---|---|
| ローディング画面 | 位置情報や天気データを取得中に表示 |
| ヘッダー | 日付と現在地を表示 |
| 天気カード | 気温、天気、体感温度などを表示 |
| 時間帯予報 | 時間ごとの天気を横スクロールで表示 |
| コーデ提案 | 天気に合った服装アイテムを表示 |
| 詳細シート | コーデの詳しい説明を表示 |
| 天気切り替え | デモ用に天気テーマを変更可能 |

### APIについて

このアプリには、OpenWeatherMap APIを使うための設定欄があります。

```javascript
const OWM_KEY = 'PASTE_YOUR_OPENWEATHERMAP_API_KEY_HERE';
```

実際の天気データを使いたい場合は、`PASTE_YOUR_OPENWEATHERMAP_API_KEY_HERE` の部分を自分のAPIキーに置き換えます。

APIキーを設定しない場合でも、アプリはデモデータで動作します。

### 使用技術

- HTML
- CSS
- JavaScript
- Geolocation API
- OpenWeatherMap API
- OpenStreetMap Nominatim
- Canvas Animation
- CSS Animation
- レスポンシブデザイン

### 特徴

このアプリは、天気情報とファッション提案を組み合わせたWebアプリです。  
背景色やアニメーションが天気によって変化するため、視覚的にも分かりやすいデザインになっています。

### 実行方法

1. `weather-outfit-app.html` をダウンロードします。
2. ブラウザでファイルを開きます。
3. 位置情報の許可を求められた場合は、許可すると現在地の天気を取得できます。
4. APIキーがない場合でも、デモモードとして利用できます。

## 共通の注意点

- どちらのファイルもスマートフォン画面を意識して作られています。
- PCでも動作しますが、スマートフォンの縦画面で見るとより自然です。
- HTML / CSS / JavaScript が1つのファイルにまとめられているため、提出や共有がしやすいです。
- 一部のフォントや天気APIを利用する場合は、インターネット接続が必要です。
- ブラウザによっては、位置情報の取得にHTTPS環境が必要になる場合があります。
